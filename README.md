## 📖 About the Project

This project showcases multiple JavaScript alert examples in a clean and attractive interface. Every button triggers a different alert, allowing users to understand how JavaScript communicates with users through pop-up messages and notifications.

Whether you're just starting your JavaScript journey or reviewing the basics, this project provides an easy-to-understand demonstration of JavaScript alerts and event handling.
